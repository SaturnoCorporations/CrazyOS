;LEIA ESTE ARQUIVO ANTES DE USAR O MELHOR SISTEMA DE TODOS!!!!

Defina a lingua desse arquivo como linguagem assembly para melhor visualização!!

O Crazy é um sistema que deve ser instalado em um dispositivo de 8Gb para baixo,
porque to com preguiça de adaptar isso :-)

Ele precisa de 1MB de ram :-),ou seja,roda até no seu microondas.

Ele usa o FileSystem FAT16 em MBR!

Ele tem 4 temas sendo eles:Purpley(Padrão), Bluey, Pinky e Dark

Comandos:
	;INICIALIZAÇÃO
	Quando o SO estiver ligando,você deve esperar...Se estiver testando algo
	você pode usar a ultima função do MENU F7,apertando o Botão Iniciar.
	Ligue apertando f7 e veja as funções:
		1-;F1 Desliga o Pc
		2-;F2 Mostra informações sobre o SISTEMA
		3-;Esc Reinicia o SISTEMA e volta.
		4-;StartButton Pula pro Desktop sem a animação,mas pode quebrar a area de trabalho.
		
	;NO DESKTOP
	Quando Estiver no Desktop aperte Ctrl+Alt para abrir o menu de segurança.
	Aperte StartButton e abra o menu Iniciar
		;No menu iniciar
			Aperte F12 e abra o PowerMenu podendo desligar ou reinciar o pc,aperte a tecla
			da função e dê enter.
